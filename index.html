Firefox Devtools Debugger

Commits to the Mozilla Firefox Web Browser Debugger:

  • QuickOpenModal

    • Update sorting algorithm to limit results to 100 sources. Update the filter function used to display sources. Update test file.

  • ShortcutsModal

    • Fix bug by preventing the console drawer from opening when the escape key is pressed to exit the ShortcutsModal component. Update CSS.

  • Sources Tabs

    • Remove blackbox from a source if a breakpoint is set in the source tab. Allow a breakpoint to be set on a source that is blackboxed. This will automatically remove black boxing from the source. Added tests to re-inforce this behavior.

  • ConditionalPanel React Component

    • Fixed bug to set and load conditions for breakpoints when Conditional Panel is opened with keyboard shortcut. Using CodeMirror API, set the shortcut to open the panel on the breakpoint that is closest to the cursor position in the Editor Component. Composed new Jest and Mochitest files to test the improved functionality of the shortcut.
